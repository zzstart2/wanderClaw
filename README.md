# 🦐 虾游 (WanderClaw)

> 替你逛互联网，把好东西寄回来。

一只有好奇心的 AI 龙虾，每天在互联网深处探险，把发现的知识宝藏写成明信片寄给你。

**OpenClaw Skill** — 安装即用，零代码，零配置。

## 安装

```bash
clawhub install wanderclaw
```

## 怎么用

虾游是**按关键词激活**的角色，不会抢主 agent 的身份。平时它在睡觉，你不找它它不冒头。

**首次使用**：跟主 agent 说一句 **"虾游"** 或 **"探索一下"**，它会：
1. 问你对什么感兴趣（2-5 个方向）
2. 注册日常 cron（每天 9/12/15/20 点自动出门）
3. 当场跑一次冷启动三连探索，给你 3 张开门红明信片

**日常触发词**（说任一个就激活）：

| 想做的事 | 触发词 |
|---|---|
| 立刻出去逛 | 虾游 / 探索 / 出发 / 找点有趣的 |
| 看历史明信片 | 最近发了什么 / 我的档案 / 明信片历史 / 统计 |
| 看单张明信片 | 详细看看 #001 |
| 要硬核深度内容 | 深潜 / 找点深的 / 深度探索 |
| 收藏喜欢的 | 收藏 #001 / 喜欢这张 |
| 屏蔽方向 | 屏蔽 NFT / 不想看 XX |
| 随机翻牌 | 随机 / 随便看看 |
| 分享到社交 | 分享 #001 |

**自动运行**（不用你管）：
- 每天 9 / 12 / 15 / 20 点自动探索，好内容推送到聊天窗
- 每周六 22:00 深潜一次，每周日 10:00 发周报
- 聊天里你提到的新兴趣会被捕获，探索方向持续个性化

## 要求

- **OpenClaw** 运行中
- **搜索工具**：至少一种（Gemini web_search 免费 / Brave / 或装个 ddg-search skill）
- **推荐模型**：Sonnet 4 或以上

## 项目结构

```
wanderClaw/
├── SKILL.md              # Skill 定义（触发规则、Onboarding、探索流程）
├── references/
│   ├── EXPLORER.md       # 探索引擎六步流程
│   ├── SOUL.md           # 虾游人格
│   ├── postcard-format.md # 明信片格式规范
│   └── sources.yaml      # 三层水域信息源
├── assets/
│   ├── state.json        # 状态模板
│   └── interest-graph.json # 兴趣图谱模板
├── scripts/
│   ├── schedule-cron.sh  # 注册定时探索 + 冷启动 cron
│   └── setup.sh          # 首次安装初始化
└── team/                 # 开发文档（BACKLOG / CHANGELOG / DECISIONS）
```

安装后，用户 workspace 会多一个 `wanderclaw/` 目录存放运行数据（明信片、探索日志、兴趣图谱等）。

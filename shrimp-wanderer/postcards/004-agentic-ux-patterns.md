# 🦐 明信片 #004

**Agentic UX：当 AI 开始替你做事，界面该怎么设计？**

阿哲，这张从 AI 产品设计方向游来。Smashing Magazine 发了篇长文，把 Agentic AI 的 UX 拆成前-中-后三阶段六模式：行动前要 Intent Preview 和 Autonomy Dial（让用户调 Agent 自主权刻度）；行动中给 Rationale 和 Confidence Signal；行动后留 Undo 和 Escalation。

核心洞察：自主权是技术输出，信任是设计输出。用户不怕 AI 做事，怕不知道它在干嘛、改不回来。

想到 #002 那张多 Agent 系统——能力越强，信任基建越关键。
